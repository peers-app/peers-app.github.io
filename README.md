# Peers Documentation

> **Live site: [peers-app.github.io](https://peers-app.github.io)**

Documentation and architecture notes for [Peers](https://peers.app/landing) — a local-first platform where your data lives on your devices, syncs peer-to-peer, and is end-to-end encrypted.

## What's here

This repo is a [Docusaurus](https://docusaurus.io/) site covering:

- **Architecture** — peer-to-peer connections, WebRTC signaling, decentralized data model
- **Packages** — building, installing, and managing Peers packages
- **Groups & members** — user/group management, sharing, access control
- **System internals** — variables, dependency injection, table access patterns

## Local development

```bash
npm install
npm start        # dev server at localhost:3000
npm run build    # production build
```
