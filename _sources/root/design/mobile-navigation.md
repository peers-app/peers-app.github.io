
# tabs on mobile and back button 

- Tabs should be ordered by the last time they were accessed, so the current tab should be at the top of the list
- The back button should close the current tab and active the next highest tab in the list
    - If there are no open tabs, it should do the current behavior which is close/minimize the app
- Double tapping the hamburger menu should quick switch to the second tab in the list (swap with the current tab, not close it)
